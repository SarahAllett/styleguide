# Styleguide
Style guide for UK Parliament

## Dependencies
* Ruby 2.3.1
* Rails 5.0.0.1
* Nodejs

## Installation

```bash
bundle install
```

```bash
cd pugin
```

```bash
npm install
```

```bash
cd ..
```

```bash
$ rails s
```
