# Styleguide
Style guide for UK Parliament

## Dependencies
* Ruby 2.3.1
* Rails 5.0.0.1
* Nodejs

## Installation

```bash
$ git submodule init
$ git submodule update
$ bundle install
$ make install -C pugin
```

To serve:

```bash
$ rails s
```
